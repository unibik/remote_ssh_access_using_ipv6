# remote_ssh_access_using_ipv6

This is a Automation tool to setup remote ssh access using IPv6

Need to know more about it and how it works, you can check my blog post [here](https://unibik.github.io/post/remote_ssh_setup/).

# Installation

```shell
sudo apt update
git clone https://github.com/unibik/remote_ssh_access_using_ipv6.git
```

# Usage
```shell
cd remote_ssh_access_using_ipv6
chmod +x remote_ssh.sh
./remote_ssh.sh
```

__Now you can access your machine through any network remotely and have fun__
